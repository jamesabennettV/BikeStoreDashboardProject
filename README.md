# BikeStoreDashboardProject
For this project, I designed a comprehensive dashboard for a bike company' executives showing what their profit per revenue is. 

~ I used SQL to pull the theoretical bike sales data from the company's database. 

~ Then I upload that data to Excel and used the data to build pivot tables & charts of each revenue type that's relevant for executives and gather them to prepare for my dashboard.

~ I send the tables that I built from Excel into Tableau to build a dynamic and interactive dashboard that executives can use to see profit per revenue. The executives are able to interact with the dashboard to show them precisely which type of revenue they need to see (per region, per year, ect).
