# BikeStoreDashboardProject
For this project, I designed a comprehensive dashboard for a bike company' executives showing what their profit per revenue is. 

~ I used SQL to pull the theoretical bike sales data from the company's database. 
~ Then I upload that data to Excel and used the data to build pivot tables & charts of each type of revenue that executives should see for my dashboard.
~ I sent the tables I built into Tableau to build a dynamic and interactive dashboard that executives can use to see profit per revenue.
