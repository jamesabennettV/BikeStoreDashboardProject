# BikeStoreDashboardProject
For this project, I designed a comprehensive dashboard for a bike company' executives showing what their profit per revenue is. I used SQL to pull the theoretical bike sales data from the company's database. Then I upload that data to excel and tableau to build a dynamic and interactive dashboard that executives can use to see profit per revenue.
